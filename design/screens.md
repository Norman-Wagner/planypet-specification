# Bildschirmaufbauten

## Dashboard
- Oben: "Hallo [Name]!" + Benachrichtigungs-Icon
- Horizontaler Avatar-Switcher (aktives Tier mit 2px blauem Rand)
- Kachel "Nächste Fütterung": Tiername, Futter, Uhrzeit, Menge, Verantwortlicher. Button [JETZT].
- Kachel "Heute im Blick": Chronologische Liste (✅ erledigt, ⏰ anstehend). Swipe links = erledigen.
- Kachel "Gesundheits-Check": Button [HAUTSTELLE ANALYSIEREN].
- Kachel "Giftköder-Warnung": Nur bei aktiver Warnung (500m).
- Kachel "FIDO-Score": Fortschrittsbalken 0-100.
- Drag & Drop aktivierbar.
- Leerzustand: Illustration + "Lege dein erstes Haustier an".

## Haustiere
- Suchfeld, Filter (Alle/Aktiv/Inaktiv)
- Tier-Karten: Avatar 56x56, Status-Punkt (grün=aktiv, grau=inaktiv)
- Buttons: [Profil] [Gesundheit] [Fotos] [Fütterung]
- FAB [+ Neues Tier] 56x56

## Community
- Top-Tabs: Feed, Betreuung, Karte
- Feed: Chronologisch, Avatar des Nutzers, ❤️ 💬 Interaktionen
- Betreuung: Angebote/Gesuche, Karten mit Bewertung, Chat-Button
- Karte: OpenStreetMap, Marker (Rot=Giftköder, Orange=Sichtung, Grün=Entwarnung)

## Profil
- Profilbild 100x100, Name, E-Mail
- Familien-Verwaltung: Rolle, Mitgliederzahl, [Mitglied verwalten], [Einladungscode]
- Einstellungen: Einheiten, Farbtheme, Glassmorphism, Benachrichtigungen, Spracheingabe, Datenschutz
- Erfolge: Grid 64x64
- [Abmelden] rot
